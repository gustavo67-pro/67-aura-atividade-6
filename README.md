# 67-aura-atividade-6
github é uma plataforma digital que salva e compartilha seus codigos com outras pessoas atraves de dados nuvem
sua principal funçao é:
compartilhar para outros usuarios posibilitando uma revissao de arquivos e codigos e colaborar em seus ajustes.
o GIT é um sistema que salva e acompanha as altaraçoes feitas nos arquivos.
ele é util para quando um grupo esta trabalhando em algo ao mesmo tempo.
para colocar seu documento online ou para publico va nas configuraçao ---> danger zone ---> change visibilit e selecione entre publico e privado
seus comandos basicos:

git add:
adiciona conteúdo do diretório de trabalho à área de preparação (ou "índice") para o próximo commit

git status: 
vai mostrar os diferentes estados dos arquivos no seu diretório de trabalho e na área de preparação. Quais arquivos foram modificados e não estão na área de preparação e quais estão na área de preparação, mas ainda não foram confirmados.

git diff:
o git diff ´usado quando voce quer ver as diferenças entre duas arvores para ver se ouve alguma alteraçao e etc...

git commit:
é usado para pegar todo o conteudo dos arquivos que foram preparados usando git add e registra um novo snapshot.

git reset:
é usado para desfazer açoes. ele move o HEAD ponteiro e , opcionalmente altera o diretorio de trabalho se usar --hard a opçao --out.

git rm:
o git rm é usado para remover arquivos, incluindo a remoçao recursica de arquivos e a remoçao de arquivos apenas da area de preparaçao mas mantendo-os no diretorio.

git mv:
 é um comando simplis para mover um arquivo e diretorio de trabalho.

 git clean:
 é um comando usado para remover arquivos indesejados do seu diretorio de trabalho. isso pode incluir a remoçao de artefatos temporarios.
